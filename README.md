### This modpack is currently under development
If you wish to test it, please install a fresh instance of 1.21.1 Neoforge and the latest version of the following mods in Mods.md

Then copy all the things from the kubejs and overrides folders, and paste them into your Minecraft instance folder. Be sure to report any features or bugs!
